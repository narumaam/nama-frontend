Auth & Security Model
Cookie Auth (Frontend)
nama_auth — HttpOnly JWT
Set via server-side /api/auth/set-cookie
Verified by jose jwtVerify in middleware
nama_demo=1 — Demo mode
SameSite=Strict
Acts as R3_SALES_MANAGER role
Hides R0/R1-only pages
Role Hierarchy (RBAC)
R0Super Admin / Founder
R1System Admin
R2_ORG_ADMINOrg Admin
R3_SALES_MANAGERSales Manager
R4_OPS_EXECUTIVEOps Executive
R5_FINANCE_ADMINFinance Admin
ABAC Conditions (JSONB)
Evaluated by can() FastAPI dep:
override_deny → always block
override_grant → always allow
roles → check role permissions
deny → default block
Conditions: geography, product_types, customer_types, deal_size, own_data_only, shift_hours, valid_until
