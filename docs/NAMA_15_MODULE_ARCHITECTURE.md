# NAMA Travel OS: 15-Module Architecture

The NAMA (Networked Autonomous Marketplace Architecture) system is composed of 15 core modules, each designed for a specific stage of the e-commerce and travel lifecycle.

2.  **M2: Itinerary Gen** - Generative AI itinerary crafting using Claude 3.5 Sonnet.
3.  **M3: Bidding Engine** - Automated supplier negotiation and dynamic bidding.

9.  **M9: Marketing OS** - WhatsApp and Email campaign management for lead nurture.
10. **M10: White-label Portals** - Multi-tenant B2B portals for travel sub-agents.

### M11–M15: Infrastructure & Intelligence
11. **M11: Corporate OS** - Enterprise-grade departure and group travel management.
12. **M12: Integration Vault** - Secure AES-256 storage for tenant API keys (Integration Hub).
13. **M13: Finance Ledger** - Automated bookkeeping and net margin reporting per tenant.
14. **M14: Sentinel Auditor** - Real-time health monitoring and self-healing engine.
15. **M15: Evolution Engine** - Recursive Self-Improvement (RSI) for agentic logic.
