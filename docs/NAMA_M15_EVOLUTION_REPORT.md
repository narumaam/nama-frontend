# NAMA Module M15: Evolution Engine (RSI)

## Overview
The **Evolution Engine** is the 15th module of the NAMA Travel OS. It implements **Recursive Self-Improvement (RSI)**, enabling the system's 10 specialized AI agents to autonomously refine their prompts and decision-making logic based on successful conversion data.

## Key Features
- **Self-Optimizing Loops:** Analyzes completed itineraries (Bali, Luxury, etc.) to identify high-conversion patterns.
- **Dynamic Weighting:** Automatically adjusts agent priority (Triage, Bidding, Itinerary) based on performance metrics.
- **Kinetic Dashboard:** A real-time monitoring interface showing learning cycles, conversion deltas, and agent accuracy.
- **RSI Bot Integration:** Feeds live success signals back into the agent prompt vault.
